
# Book My Show Backend Application

•	Book My Show is a movie ticket booking system project.  
•	Its a backend code written using Spring-Boot.          
•	It consists of Different Layers like Controller, Service,Repository , DTO(Data Transfer Object)              
•	These layers in the project increases its code maintainability, understandability and readability.



## Functionality

• Add a Movie.                                            
• Add a Theater.                                      
• Add Theater Seats            
• Create a Show for any Movie in any theater  
• Create Show Seats for booking purpose
## Different Models / Entities / (Table in MySQL) In the Project

• User Entity  
• Movie Entity  
• Theater Entity  
• Show Entity  
• Theates Seat Entity  
• Show Seat Entity  
• Ticket Entity  
## Teck-Stack Used :

• Spring-Boot  
• MySQL  
• Java  
• Hibernate  
• JPA
